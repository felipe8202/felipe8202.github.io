Guia de Referência Rápida — Desenvolvimento Web

1. Roteiro de Estudos

Etapa 1 — Fundamentos:

  1. Estrutura Básica do HTML
    1. <!DOCTYPE html>
    2. <html>
    3. <head>
    4. <title>
    5. <body>
  2. Elementos e Tags Fundamentais
    1. <h1> a <h6> - Títulos e subtítulos (hierarquia de importância).
    2. <p> - Parágrafos de texto.
    3. <a> - Links para outras páginas ou recursos.
    4. <img> - Exibição de imagens.
    5. <div> e <span> - Containers genéricos para organização e estilização.
  3. HTML e semântica
    1. Atributos globais (ex: <span class="exemplo">, <div id="exemplo">).
    2. <img alt="descrição"> - Descrição alternativa importante para acessibilidade.
    3. <a href="https://exemplo.com"> - Define o destino do link.
    4. Indentação e organização do código - Facilita leitura e manutenção.
    5. Semântica (ex: <header>, <footer>, <article>).

Etapa 2 — Estilização:

  1. Fundamentos Visuais
    1. Cores: utilização planejada de paletas cromáticas para fortalecer a identidade visual e orientar a percepção do usuário.
    2. Tipografia: escolha adequada de fontes, tamanhos e pesos para garantir legibilidade, hierarquia e clareza na comunicação.
    3. Espaçamento: aplicação equilibrada de margens, preenchimentos e intervalos para melhorar a organização dos elementos na interface.
    4. Hierarquia visual: definição da importância dos conteúdos por meio de contraste, escala, posição e destaque visual.
    5. Consistência visual: manutenção de padrões gráficos para tornar a interface mais coerente, profissional e fácil de compreender.
  2. Estrutura e Layout
    1. Grid: recurso utilizado para organizar elementos em linhas e colunas, permitindo maior controle sobre a composição da página.
    2. Flexbox: modelo de layout indicado para alinhamento e distribuição de elementos em uma dimensão, seja horizontal ou vertical.
    3. Responsividade: capacidade da interface de se adaptar a diferentes tamanhos de tela, mantendo funcionalidade e boa apresentação.
    4. Alinhamento: organização dos elementos com base em eixos visuais, contribuindo para equilíbrio, leitura e harmonia da página.
    5. Proporção dos elementos: definição adequada de larguras, alturas e limites para evitar distorções e melhorar a composição visual.
  3. Componentes e Interatividade
    1. Botões: elementos de ação que devem apresentar clareza visual, destaque adequado e comportamento previsível para o usuário.
    2. Cards: estruturas visuais utilizadas para agrupar informações relacionadas de forma organizada, objetiva e acessível.
    3. Imagens: recursos visuais que devem ser dimensionados corretamente, preservando qualidade, proporção e coerência com o conteúdo.
    4. Animações: efeitos visuais aplicados para tornar a navegação mais dinâmica, desde que utilizados com moderação e finalidade clara.
    5. Estados visuais: variações de aparência em elementos interativos, como hover, focus e disabled, que auxiliam na usabilidade da interface.

2. Ranking de Tecnologias

  1. Node.js

    É um ambiente de execução que permite utilizar JavaScript no lado do servidor, possibilitando a criação de APIs, sistemas web, autenticações, conexões com banco de dados e outras funcionalidades de back-end.

  2. React

    É uma biblioteca JavaScript voltada para a criação de interfaces modernas, organizadas em componentes reutilizáveis, facilitando o desenvolvimento e a manutenção de aplicações web.

  3. CSS

    É a tecnologia utilizada para estilizar páginas web, permitindo controlar cores, fontes, espaçamentos, tamanhos, alinhamentos, layouts e a adaptação da interface a diferentes telas.

  4. HTML

    É a linguagem de marcação responsável por estruturar o conteúdo de uma página web, definindo elementos como títulos, parágrafos, imagens, links, listas e seções.

  5. JavaScript

    É uma linguagem de programação usada para adicionar interatividade às páginas, como menus dinâmicos, validações de formulários, animações e atualização de conteúdos sem recarregar a página.

3. Glossário Técnico

HTML
Linguagem de marcação utilizada para estruturar o conteúdo de páginas web.
Base da construção de documentos na internet, definindo elementos como títulos, parágrafos, links e imagens.
CSS
Tecnologia responsável pela estilização visual das páginas web.
Permite controlar cores, fontes, espaçamentos, layouts e responsividade da interface.
Front-end
Interface do usuário
Parte visual e interativa de um sistema web com a qual o usuário tem contato direto.
Back-end
Lado do servidor
Parte responsável pela lógica, processamento de dados, autenticação e comunicação com bancos de dados.
DOM
Modelo de representação da estrutura de uma página web, permitindo que linguagens como JavaScript acessem e modifiquem elementos HTML.
API
Conjunto de regras e recursos que permite a comunicação entre diferentes sistemas, aplicações ou serviços.
HTTP
Protocolo utilizado para a comunicação entre navegadores e servidores na transferência de páginas e recursos da web.
URL
Endereço utilizado para localizar um recurso na internet, como uma página, imagem, arquivo ou serviço online.
