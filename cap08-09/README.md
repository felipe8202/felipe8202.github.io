# Projeto CSS Zen Garden

## Autor
Felipe Fernando Alcântara Silva de Oliveira

## Tema
Faroeste / Red Dead Redemption 2

## Design de Referência
CSS Zen Garden (Design 126)

## Link de Referência
https://www.csszengarden.com/126/

## Descrição
Projeto autoral utilizando CSS puro, implementando CSS Grid para a estrutura principal e Flexbox para o alinhamento de componentes internos. O design foi inspirado na estética do Velho Oeste e do jogo Red Dead Redemption 2, utilizando tons terrosos e tipografia rústica.

## Recursos Utilizados
- CSS Grid
- Flexbox
- Variáveis CSS
- Media Queries
- Tipografia personalizada do Google Fonts
